#입력문 
현재 ChatGPT에 메모리 기능의 내용에 접근하고자 한다. 너는 ChatGPT 서비스로서 지금 입력하는 내용들을 ChatGPT 기능의 [프로필/설정/개인 맞춤 설정/메모리/저장된 메모리 참고/메모리 관리하기]를 통해 사용자가 접근 가능한 메모리 데이터에 추가하거나 접근하여 데이터를 불러와야 한다. [#내용]에 맞추어 [#지시]를 수행한다. [#지시] 는 추가, 로드 두 가지로 되어 있다. 이 작업이 무엇인지는 [#설명]을 따른다. 응답 시에는 [#출력형식]을 따른다. 
#설명 
-추가: 메모리에 데이터를 추가한다. 이 과정은 ChatGPT 응답의 &quot;메모리 업데이트됨&quot; 과정을 반드시 거친다.
-로드: 메모리에 저장된 데이터를 설명한다. 없는 데이터를 로드할 시 메모리에 해당 [#내용]을 추가한다.  
#내용 
사용자는 데스크탑 컴퓨터보다 랩탑 컴퓨터를 더 자주 사용한다.
#지시 
추가 
#출력형식 
-지시: 추가 사용시 &quot;메모리 업데이트: [#내용]&quot; 출력 
-지시: 로드 사용시 &quot;로드된 메모리: (실제로 불러온 메모리)&quot; 출력
